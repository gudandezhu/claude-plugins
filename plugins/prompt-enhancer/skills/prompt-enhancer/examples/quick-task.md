# Example 1: Quick Task (R.T.F Framework)

## Original Prompt

```
write a python function to calculate fibonacci
```

## Enhanced Prompt

```
Role: You are a Python developer with expertise in algorithms and efficient code.

Task: Write a Python function to calculate Fibonacci numbers. The function should:
- Take an integer n as input
- Return the nth Fibonacci number
- Include error handling for invalid inputs
- Be efficient (consider memoization or iterative approach)

Format: Provide the complete function with:
1. Clear docstring explaining the algorithm
2. Type hints for parameters and return value
3. Example usage in a if __name__ == "__main__" block
4. Brief comment on time and space complexity
```

## Analysis

**Task Type**: Quick task / Technical
**Framework Selected**: R.T.F (Role - Task - Format)
**Rationale**: Simple, well-defined request that benefits from clear structure

**Improvements Made**:
- Added specific Role to establish expertise context
- Expanded Task with clear requirements (error handling, efficiency)
- Defined Format for complete, production-ready output
- Added expectations for documentation and examples
