# Agile Start - 启动敏捷开发流程

## 任务

清除暂停标记，触发 agile-start 技能完成初始化和任务管理。

---

## 执行流程

### 第一步：清除暂停标记

```bash
# 删除暂停标记（如果存在）
if [ -f "projects/active/pause.flag" ]; then
    echo "🔄 清除暂停标记，启用自动继续模式"
    rm -f projects/active/pause.flag
fi
```

### 第二步：触发 agile-start 技能

此命令只负责清除暂停标记，剩余工作由 agile-start 技能自动完成：
- 项目初始化（首次使用）
- 加载项目状态
- 获取下一个待执行任务
- 引导用户添加任务或开始开发

---

```markdown
- 加载项目状态
- 准备下一个任务

💡 添加任务: p0: 实现新功能
```

---

## 注意事项

1. **命令职责**: 只负责清除暂停标记和触发技能
2. **技能驱动**: 所有初始化和任务管理由技能自动完成
3. **随时添加任务**: 使用 `p0: 任务描述` 格式随时添加任务
