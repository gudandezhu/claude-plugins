# 需求
1. 优化claude-bot插件，插件仅需要start和stop命令提供给用户，其他过程由skill完成
2. 在迭代过程中，允许用户随时添加任务，并指定优先级，例如 p0:新增需求
3. 自动任务管理：
- 跟踪任务进度
- 根据优先级排序任务
- 检查任务是否完成
- 决定下一步做什么
4. 需求迭代文档维护在ai-docs下，且启动插件时候，.gitignore内添加ai-docs

5. 维护以下文档

   | 文档          | 用途                           |
   | ------------- | ------------------------------ |
   | PRD.md        | 项目需求文档                   |
   | PLAN.md       | 工作计划和任务清单             |
   | OPS.md        | 操作指南（启动、构建、测试等） |
   | CONTEXT.md    | 项目上下文和记忆               |
   | ACCEPTANCE.md | 任务验收报告                   |
   | BUGS.md       | 发现的 Bug 列表                |
   | API.md        | 项目的 API 清单                |

6. 当完成任务后，自动做以下操作
   - 自动运行测试
   - 检查是否达到验收标准
   - 记录测试结果
   - 发现问题会自动修复或报告



### 典型工作流程

1. **开发过程中**

- 告诉 插件 你要做什么
- 插件 会自动更新计划
- 插件 会记录进度和发现

1. **遇到问题**

- 插件 会自动诊断
- 插件  尝试自动修复
- 插件  记录到 BUGS.md

1. **完成任务**

- 插件 会自动测试
- 插件 更新验收报告
- 插件 开始下一个任务
