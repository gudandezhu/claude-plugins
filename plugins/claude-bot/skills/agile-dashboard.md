# Agile Dashboard - 进度展示

## 任务

生成简洁的项目状态报告，帮助用户快速了解当前进度。

---

## 执行流程

### 第一步：读取项目状态

从 `projects/active/iteration.txt` 读取当前迭代编号，然后读取 `projects/active/iterations/{iteration}/status.json` 获取状态数据。

### 第二步：显示进度信息

使用 jq 解析状态文件，显示：
- 迭代编号
- 任务完成进度（已完成/总数）
- 测试覆盖率
- 当前任务
- 待办任务列表（最多显示 3 个）
